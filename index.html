O que é cloud computing?

É um termo utilizado para se referenciar a utilização de recursos computacionais através da internet (nuvem).

Como surgiu essa ideia?

Time-sharing

Remote job entry

É enviada uma tarefa de processamento não interativo para um mainframe a partir uma estação de trabalho remota.

Quando o processamento é finalizado, ele retorna o resultado para a estação de trabalho.

Cloud computing

"Cloud computing é um paradigma da computação onde os limites computacionais estão determinados pelo custo, não pelos limites técnicos."

Tipos de cloud computing

IaaS (Infrastructure as a Service)
SaaS (Software as a Service)

Modelos de implantação de Infrastructure as a Service

Nuvem Pública

Infraestrutura compartilhada por várias organizações.

Os recursos pertencem a um provedor terceirizado e são operados por ele.

Nuvem Privada

A infraestrutura é organizada pela organização que está contratando ela.

Pode ser on-premise ou off-premise.

Nuvem Híbrida

Combinação de nuvens públicas e privadas, com portabilidade de dados e aplicativos entre elas.
